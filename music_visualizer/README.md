# Professional Music Visualizer

A powerful Windows application for creating stunning music visualizations with professional effects, video overlays, and extensive customization options.

## Features

- **Multiple Visualizer Types**: Waveforms, frequency bars, circular spectrum, particle effects
- **Special Effects**: Sparkles, glistening water effects, glow, blur
- **Video & Image Support**: Overlay visualizers on videos or use images as backgrounds
- **Audio Format Support**: MP3, WAV, FLAC, OGG, and more
- **Advanced Controls**:
  - Color customization with opacity
  - Position and size adjustment
  - Fade in/out effects
  - Timing controls
  - Multiple visualizers simultaneously
- **Export**: Render to high-quality video (MP4)

## Installation

1. Install Python 3.8 or higher
2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
python main.py
```

### Quick Start

1. **Load Audio**: Click "Load Audio" and select your music file
2. **Add Visualizers**: Choose from various visualizer types
3. **Customize**: Adjust colors, positions, sizes, and effects
4. **Preview**: See your creation in real-time
5. **Export**: Render to video file

### Background Options

- **No Background**: Pure visualizations on black/transparent
- **Color Background**: Solid color or gradient
- **Image Background**: Static image backdrop
- **Video Background**: Overlay visualizers on video

## Controls

- **Play/Pause**: Space bar
- **Seek**: Click on timeline
- **Add Visualizer**: Click visualizer type buttons
- **Customize**: Select visualizer and adjust properties
- **Export**: Configure output settings and render

## Visualizer Types

1. **Waveform**: Classic oscilloscope-style audio waveform
2. **Frequency Bars**: Vertical bars responding to frequency bands
3. **Circular Spectrum**: Radial frequency visualization
4. **Particles**: Beat-reactive particle system
5. **Radial Lines**: Rotating lines synced to music
6. **Glow Orbs**: Pulsing orbs following the beat

## Effects

- **Sparkles**: Twinkling particles on beats
- **Water Ripples**: Glistening water-like effects
- **Glow**: Soft glow around visualizers
- **Motion Blur**: Smooth trailing effects
- **Color Cycling**: Automatic color transitions

## System Requirements

- Windows 10 or higher (also works on Linux/Mac)
- Python 3.8+
- 4GB RAM minimum (8GB recommended)
- Dedicated GPU recommended for smooth rendering
